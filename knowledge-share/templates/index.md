# ナレッジ・インデックス

このファイルは `~/.claude/CLAUDE.md` から `@~/.claude/knowledge/index.md` で
全リポジトリ・全セッションに自動読み込みされます（Claude Code の @import 機能）。

**運用ルール（公式の自動メモリに準拠）**

- 1エントリ＝1行。本体は `topics/<topic>.md` に置き、ここはタイトルと参照だけ。
- このファイルは **200行 / 25KB 以内**に保つ（自動メモリの予算と同じ）。
- 超過したら古いエントリ行を `index-archive.md` に退避する（topics 側の本体は残す）。
- 詳細が必要になったら、該当 topic ファイルをオンデマンドで開く。

記録は `/kb`、検索は `/kb search <語>`、過去会話からの採掘は `/kb-harvest`。
